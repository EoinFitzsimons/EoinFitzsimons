# Image filenames for /assets

# Avatar (profile photo)
avatar.jpg  - Your personal photo for the avatar in the hero section

# Project cover image
project-cover.png  - Cover image for the Selected Project section

# Open Graph / social preview image
og-image.png  - Used for social sharing and meta tags

# Game thumbnails (hosted in each game's repo, not in assets)
Each game should have a file named 'thumbnail.png' in the root of its GitHub Pages repository.
The site will load the thumbnail for each game from:
  https://eoinfitzsimons.github.io/<repo>/thumbnail.png

Example game repos:
- Eoin-s-Building-Game/thumbnail.png
- Eoin-s-Football-Game/thumbnail.png
- Eoin-s-Text-Game/thumbnail.png
- Eoin-s-Racing-Game/thumbnail.png
- Skyworks-Concourse/thumbnail.png
- Rith-Dearg/thumbnail.png
